Welcome to ZapTec Reporting

Welcome to the ZapTec Reporting system. This is a simple web application that allows you to generate reports based on your data.

To get started, please select a report type from the menu above.

ZapTec Reporting is run from a container and when left running it will daily gather all data from the past 365 days. Should you somehow lose the database or corrupt the database, just remove the database file and restart the container. The system will then recreate the database and start gathering data from the last 365 days again. On the 'Get Data' page you also have an option to force read the past 365 days. The ZapTec API limits the history to 365 days.

When the container is running and reads the data at the same day in the month as found in the .env var "reportday", it will generate a report for the previous month. The report will be sent to the email address found in the .env var "reportemail".

You can also generate your own report using the 'reports' option and export this to excel.

The tarif used is also in the .env file and is not in dollars or Euro or other currency for ease of use.
